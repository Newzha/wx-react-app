基于React+Ant Design（antd、ant-design-pro）全家桶的框架整合，集合了react-redux、react-router-redux、redux-saga、axios等技术。  
## 安装
下载完成后，在项目根目录使用 `npm install` 或者 `yarn install` , 推荐使用 `yarn` 安装。
## 启动
在根目录运行如下命令:  
`yarn start`
